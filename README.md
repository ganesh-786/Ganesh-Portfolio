## 🚦 Getting Started

To run the portfolio locally:

1. **Clone the Repository**
    ```bash
    git clone https://github.com/ganesh-786/Ganesh-Portfolio.git
    cd Ganesh-Portfolio
    ```

2. **Open `index.html` in your browser**
    ```bash
    # On most systems, you can simply double-click index.html
    # Or use:
    open index.html
    ```
